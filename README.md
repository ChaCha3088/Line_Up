### Summary
> #### 대학교 축제 주점 부스에서 간단하게 사용할 수 있는 POS, 신청곡 게시판, 자유게시판 서비스 입니다.

### Function

+ 간단한 POS 기능
  + 직원은 메뉴와 수량, 합계 등을 간편하게 확인하고 추가, 수정, 삭제 가능

+ 신청곡 게시판 기능
  + 원하는 음악의 아티스트와 제목을 신청할 수 있음

+ 자유게시판 기능
  + 대학 축제에서 같은 주점을 이용하는 사람은 회원가입 후 게시판을 통해 자유롭게 대화 가능
